Elements of A Computer System

  • A computer is an electronic device capable of performing commands
    • Basic commands that a computer is an input (gets data), output (display result), storage, and performance of arithmetic and logical operations
  • Two main components of a computer system are hardware and software

Hardware

  • Major hardware components include:
    • The central processing unit(CPU)
    • Main memory (MM), also known as the random-access memory (RAM)
    • Input Devices (Keyboard, Mouse, Secondary Storage, etc.)
    • Output Devices (Screen/Monitor, printer, secondary storage, etc.)

Software

  • Software are programs written to perform specific tasks
    • Ex). Word processors are programs that you use to write letters, papers, and even books
    • All software is written in programming languages

The Language of a Computer

  • Electrical signals are used inside the computer to process information
    • The two types of electrical signals include:
      • Analog
        • Analog signals are continuously varying continuous waveforms used to represent such things as sound
          • Audio tapes, for example, store data in analog signals
      • Digital
        • Digital signals represent information with a sequence of 0s and 1s
          • A 0 represents a low voltage and a 1 represents a high voltage
