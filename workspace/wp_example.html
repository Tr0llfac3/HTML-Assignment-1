EXAMPLE 1-1

In this example, we design an algorithm to find the perimeter and area of a
rectangle.

To find the perimeter and area of a rectangle, you need to know the rectangle’s
length and width

The perimeter and area of the rectangle are then given by the following formulas:

Perimeter = 2 * (length + width)

Area = length * width

The algorithm to find the perimeter and area of the rectangle is:

  • Get the length of the rectangle.
  • Get the width of the rectangle.
  • Find the perimeter using the following equation:
    • Perimeter = 2 * (length + width)
  • Find the area using the following equation:
    • Area = length + width

sales tax, the city’s portion of the sales tax, and, if it is a luxury item, the luxury tax.
